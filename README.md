# Ledger

A C# personal accounting application / 一个 C# 个人账本管理应用

[中文](#中文-chinese) | [English](#english)

---

## English

### Overview

Ledger is a personal accounting management system built with C#. It provides a simple and efficient way to track and manage your daily transactions and financial records.

### Features

- 📝 **Transaction Recording** - Record income and expenses
- 📊 **Data Management** - Organize and manage financial data  
- ✅ **Testing** - Comprehensive unit test coverage
- 🔍 **Code Analysis** - Integrated code quality analysis

### Tech Stack

- **Language**: C#
- **Framework**: .NET
- **Testing**: Unit tests included
- **IDE**: Visual Studio / VS Code

### Quick Start

```bash
# Clone the repository
git clone https://github.com/Anfellez/Ledger.git
cd Ledger

# Build the project
dotnet build code/ledger.sln

# Run tests
dotnet test code/ledger.Tests
```

### Project Structure

```
code/
├── ledger/              # Main application
├── ledger.Tests/        # Unit tests
└── ledger.TestsRunner/  # Test runner
```
